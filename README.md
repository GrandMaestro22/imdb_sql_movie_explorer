# IMDb Movie Database Explorer

A beginner-friendly SQL project exploring the IMDb dataset using SQL queries.  
Built using DataGrip + the IMDb Complete Dataset (Kaggle).

## Features
- Top-rated movies
- Movies by genre
- Movies by actor
- Popular movies by vote count
- Directors of a movie
- Average rating by genre

## Technologies
- SQL
- DataGrip
- SQLite

## How to Use
1. Import the IMDb CSV files into your database.
2. Run the queries in `queries.sql`.
3. Explore and modify the queries to learn SQL.

## Dataset
IMDb Complete Dataset (Kaggle)
